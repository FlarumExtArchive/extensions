# Flarum Extensions Archive
Inspired by the loss of `the-turk` and the discussion on [discuss.flarum.org/38631](https://discuss.flarum.org/d/38631), a project to maintain mirrored repositories of public Flarum extensions.

Extensions are added to a schedule to maintain a mirror in private repositories, if it is lost the repository will then be made public. This is maintain the primacy of the authors repository and provide a read only copy for those who wish to fork (and hopefully develop & maintain) or those in need of a archived copy (eg, need to rebuild for some reason).
